## ICEスコア評価フレームワーク

新規施策・案件を以下で評価する（各軸1〜10点、合計30点満点）：

| 軸 | 評価基準 |
|----|---------|
| **Impact（影響度）** | ノーススターの収益・ブランド・クライアント満足への影響 |
| **Confidence（確信度）** | 施策が成功する確信の高さ（データ・事例があるか） |
| **Ease（実施容易性）** | 1人経営・AI組織で実行できるか（時間・コスト・技術） |

**判定：24点以上 → 今すぐやる / 15〜23点 → 次の四半期 / 14点以下 → 保留**

---

## 競合分析フレームワーク

競合を以下の4軸で評価する：
1. **価格帯**：月額料金・初期費用
2. **AI活用度**：自動化・24時間対応の有無
3. **介護専門性**：処遇改善加算・介護報酬対応の深さ
4. **顧客獲得チャネル**：SNS・SEO・紹介・訪問営業

## 実行手順（この順番を守れ）

### 新規案件評価
1. ICEスコアで数値評価する（感覚評価禁止）
2. 競合4軸で差別化ポイントを確認する
3. 月次収益モデルを試算する（BUN_CEO確認前）
4. BIZ_評価レポートをType E形式で作成してDriveに保存する
5. BUN_CEOに提案する（金額・契約条件はBUN_CEOが決定）

### SNSコンテンツ方針立案
1. MKT_PRタイムズWFの出力（PRタイムズスキャン）から業界トレンドを確認する
2. ターゲット（介護施設経営者・グループホーム管理者）の課題に引き寄せる
3. 具体的な数字・事例を必ず含める（「一般論」は禁止）
4. MKT_SNS WFのシステムプロンプトに反映する場合はn8n APIで更新する
