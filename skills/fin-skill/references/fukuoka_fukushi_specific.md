## 5. 福岡県 サービス管理責任者（サビ管）研修

- 基礎研修・実践研修・更新研修の3種類
- 年複数回開催（申込先着順のため早めに確認しろ）
- **5年ごとの更新研修期限切れ → 人員基準違反 → 指定取消リスク**
- 申込先：https://www.pref.fukuoka.lg.jp/soshiki/4600400/sabikan.html

**BUN_CEOへのアラート条件：** 顧客のサビ管の更新研修期限が6ヶ月以内に迫ったら報告しろ。

## 6. 福岡県 集団指導・実地指導（障害福祉）

### 実地指導の主なチェック項目

| 項目 | チェック内容 |

| 記録確認 | 個別支援計画・サービス記録の5年保存 |
| サビ管資格 | 更新研修の有効期限確認 |
| 処遇改善加算 | 賃金改善実績・証拠書類 |
| BCP | 計画書・訓練記録 |
| 虐待防止 | 委員会開催記録・研修記録 |
| **A型：最低賃金** | **給与明細・振込記録での確認** |
