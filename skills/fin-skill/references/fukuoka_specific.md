## ⚠️ 管轄窓口（最重要）

| 種別 | 所管部署 |
|------|---------|
| 居宅サービス・特定施設の指定 | 福岡県 健康福祉部 地域福祉課 / 各保健福祉環境事務所 |
| 処遇改善加算の申請 | 同上 |
| 有料老人ホームの届出 | 高齢者地域包括ケア推進課 |
| 介護保険審査委員会（苦情） | 福岡県介護保険審査委員会 |
| 集団指導 | 各保健福祉環境事務所（年1回） |
| 実地指導 | 同上（不定期・通常2〜3年に1回） |

## 1. 福岡県 指定基準条例（特定施設）

| 項目 | 国の基準 | 福岡県条例 | 差異 |
|------|---------|----------|------|
| **記録保存期間** | 2年 | **5年** | ⚠️ 福岡は長い |
| 苦情対応記録 | 規定あり | 同等 | — |
| 居室面積 | 13㎡以上 | 同等（要確認） | — |

公式確認先：https://www.pref.fukuoka.lg.jp/contents/kaigo-hoken.html

---

## 2. 福岡県 処遇改善加算（申請手続き）

| 項目 | 内容 |
|------|------|
| 計画書提出締切 | **2月末日（必着か消印有効か毎年要確認）** |
| 提出方法 | 郵送または電子申請（年度ごとに変わる） |
| 書式 | 県指定の書式を使え（国の様式と異なる場合あり） |
| 実績報告 | 翌年度7月31日 |

問い合わせ：地域福祉課 居宅サービス班 092-643-3260（代表・要確認）

---

## 3. 福岡県 集団指導

- 年1回、参加義務あり（代理出席可）
- 欠席すると実地指導の対象になりやすい
- 日程・資料：https://www.pref.fukuoka.lg.jp/contents/kaigo-hoken.html

---

## 4. 福岡県 実地指導（主なチェック項目）

| 項目 | チェック内容 |
|------|------------|
| 記録確認 | サービス記録・ケアプラン・業務日誌の**5年分**保存 |
| 人員配置 | 勤務実績（タイムカード等）と届出の整合性 |
| 処遇改善加算 | 賃金改善実績・職場環境等要件の証拠書類 |
| BCP | 計画書の存在・訓練記録 |
| 身体拘束 | 記録の適切性・家族同意書 |

**実地指導通知が届いたら即BUN_CEOに報告しろ。**

---

## 5. 福岡県 地域区分（介護報酬の地域加算）

| 地域 | 区分 | 上乗せ率 |
|------|------|---------|
| 福岡市 | 4級地 | 10% |
| 北九州市 | 5級地 | 6% |
| 久留米市・春日市等 | 6級地 | 3% |
| その他市町村 | 7級地 | 0% |

⚠️ **顧客の事業所所在地の区分を必ず確認しろ。単価計算に直結する。**
