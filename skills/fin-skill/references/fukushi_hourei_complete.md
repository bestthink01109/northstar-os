## 主要法令体系

| 法令 | 内容 |
|------|------|
| **障害者総合支援法** | 障害福祉サービス全般の根拠法 |
| **児童福祉法** | 障害児支援（放課後等デイ・児童発達支援等） |
| **障害者雇用促進法** | 就労継続支援A型・就労移行支援と連動 |
| **障害者虐待防止法** | 虐待防止・通報義務 |
| **障害者差別解消法** | 合理的配慮の義務化（2024年改正で民間も義務） |
| **個人情報保護法** | 利用者情報管理 |

## 1. サービス種別と報酬体系

### 就労系

| サービス | 対象 | 特記事項 |
|---------|------|---------|
| **就労継続支援A型** | 雇用契約を結べる65歳未満の障害者 | **最低賃金を必ず払う。違反は指定取消** |
| **就労継続支援B型** | 雇用契約困難な障害者 | 工賃支払義務（最低賃金不要） |
| **就労移行支援** | 就労を目指す65歳未満の障害者 | 就職率・定着率で報酬が変動 |

### 居住・日中活動系

| サービス | 対象 |
|---------|------|
| **共同生活援助（グループホーム・障害者）** | 障害者 |
| **生活介護** | 常時介護が必要な障害者（主に区分3以上） |
| **自立訓練** | 身体・生活能力訓練が必要な障害者 |

### 障害児支援

| サービス | 対象 |
|---------|------|
| **放課後等デイサービス** | 就学中の障害児（6〜18歳） |
| **児童発達支援** | 未就学の障害児 |

**改定タイミング：** 原則3年ごと（次回2027年度予定）

## 3. 人員配置基準

### 就労継続支援A型・B型（共通）

| 職種 | 基準 |
|------|-----|
| 管理者 | 1名 |
| **サービス管理責任者（サビ管）** | 利用者60人に1名以上（**5年ごと更新研修必須**） |
| 職業指導員 | 利用者10人に1名以上（常勤換算） |
| 生活支援員 | 利用者10人に1名以上（常勤換算） |

### 放課後等デイサービス

| 職種 | 基準 |
|------|-----|
| 管理者 | 1名 |
| 児童発達支援管理責任者 | 1名以上（**5年ごと更新研修必須**） |
| 指導員または保育士 | 利用者10人に2名以上（うち1名常勤） |

### グループホーム（障害者）

| 職種 | 基準 |
|------|-----|
| 管理者 | 1名 |
| サービス管理責任者 | 利用者30名以下は1名 |
| 世話人 | 利用者4〜5人に1名 |

## 4. 運営基準（共通）

| 基準項目 | 内容 |
|---------|------|
| 個別支援計画 | サービス開始時・6ヶ月ごとに作成・見直し |
| **記録保存** | **5年間**（介護と同様） |
| 苦情処理 | 受付窓口設置・記録保存 |
| 身体拘束禁止 | 緊急性・非代替性・一時性の3要件 |

### 就労継続支援A型 特有

| 項目 | 内容 |
|------|------|
| **最低賃金の支払い義務** | 雇用契約があるため最低賃金法が適用 |
| 労働保険（雇用保険・労災） | 全員に適用 |
| 36協定 | 時間外労働がある場合は締結必須 |
| 平均工賃額の公表 | 年1回公表義務 |

⚠️ **最低賃金未払いが発覚した場合は即座にBUN_CEOに報告しろ。指定取消につながる。**

## 6. 虐待防止【障害者虐待防止法】

### 虐待の5類型
1. 身体的虐待
2. 性的虐待
3. 心理的虐待
4. 放棄・放置（ネグレクト）
5. 経済的虐待

### 事業者の義務

| 義務 | 内容 |
|------|------|
| 通報義務 | 虐待（疑い含む）を発見したら市区町村に通報 |
| 虐待防止委員会 | 設置・定期開催 |
| 研修 | 年2回以上 |
| 責任者設置 | 虐待防止のための責任者を配置 |

---

## 7. 差別解消【2024年改正・民間も義務化】

| 項目 | 内容 |
|------|------|
| 不当な差別的取扱い | 禁止 |
| **合理的配慮の提供** | **義務**（2024年4月から民間も） |

---

## 8. サービス管理責任者（サビ管）の資格管理

| 要件 | 内容 |
|------|------|
| 実務経験 | 対象業務で3〜5年以上（区分による） |
| 基礎研修 | 修了必須（8時間程度） |
| 実践研修 | 修了必須（24時間程度） |
| **更新研修** | **5年ごとに必須**（期限切れ → 人員基準違反） |

⚠️ **サビ管の研修期限管理は必ずやれ。切れると即人員基準違反になる。**
