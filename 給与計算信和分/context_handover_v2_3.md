## 2. auto_input.py の機能と処理フロー

### 処理フロー（1人分）

```
給与明細書一覧（マウスホイール上で先頭スクロール → Page Downで対象行へ）
  → 「開く」クリック
    → 給与明細書画面
      → 「勤怠」クリック
        → タイムカード入力画面
          → 「設定」クリック
            → パターン選択（A〜E）
            → 早出/遅刻タブ → 遅刻早退チェックボックスON/OFF（ピクセル判定）
            → 集計時間タブ → 休憩時間の「時」部分をダブルクリックして上書き
            → OKクリック
          → データ入力（座標クリック方式、3ページ: 1-12日/13-24日/25-31日）
          → 「明細へ書き込み」クリック（自動で明細書画面に戻る）
        → 「更新」クリック
        → 「閉じる」クリック（一覧に戻る）
  → 次の従業員へ
```

### 主要機能

- 座標クリック方式で高速入力（Tab方式から変更済み）
- チェックボックスのピクセル色判定（ON/OFF自動判別）
- Escキーで緊急停止（keyboardモジュール）
- エラーダイアログ検出で自動停止（pygetwindow）
- マウスを左上隅に移動でもフェイルセーフ停止
- 有給/欠勤時はメモ欄に「有給」「欠勤」を自動入力
- 時給者の有給時間を自動入力
- 一覧スクロール: マウスホイール上→Page Downで対象行へ

### 実行方法

```
cd %USERPROFILE%\OneDrive\デスクトップ\timecard
python auto_input.py 202603
```

## 4. タイムカード入力画面の仕様

### ページング

- 画面に13行表示
- Page Downで最終行が先頭に移動（オーバーラップ方式）
- ページ分割: 1-12日(ページ1) → PgDn → 13-24日(ページ2) → PgDn → 25-31日(ページ3)
- 各ページ内でrow_idxが0からリセット

### 給与明細書一覧のスクロール

- マウスホイール scroll(200) で先頭に戻す
- Page Downで1ページ=14行移動（オーバーラップなし）
- index // 14 = Page Down回数、index % 14 = 画面内の行番号
- 1スクロール（マウスホイール1ノッチ）= 約7行移動

### 設定ダイアログ

- 勤怠パターンは全従業員共通設定（パターンC全体に影響等）
- そのため各従業員処理前に毎回設定変更を実施
- 休憩時間: ダブルクリックで「時」部分だけ選択して上書き（分は常に00）
- Ctrl+Aは使えない、トリプルクリックも使えない

### 入力値の仕様

- 時刻はコロンなしで入力（例: 8:30 → 830）
- 日本語（有給、欠勤）はクリップボード経由ペースト
- チェックボックスはSpaceキーでトグル

## 6. CSVファイル仕様

### ファイル名規則

`社員コード_氏名.csv`

- 例: 社員8_池田秦幸.csv, パート8_松井美香.csv, 社員15_松井美香.csv
- 社員コードで一意にマッチ（同姓同名対策）
- コードなしの場合: 松井慶信.csv（名前でマッチ）

### CSV列構成（BOM付きUTF-8で保存必須）

```
日付,出勤時間,退出時間,有給,欠勤,有給時間
```

### データ例

```csv
3月2日,8:30,18:45,,,          ← 通常出勤
3月14日,,,1,,                 ← 有給（月給者）
3月14日,,,1,,4:00             ← 有給（時給者・有給時間入力）
3月8日,,,,1,                  ← 欠勤
```

### BOM付きUTF-8での保存方法（Python）

```python
with open(filepath, 'w', encoding='utf-8-sig') as f:
    f.write('日付,出勤時間,退出時間,有給,欠勤,有給時間\n')
    # ... データ行
```

## 7. タイムカードOCR → CSV変換仕様

### タイムカード形式

- MAX社 ER-Sカード
- 前半カード: 1日〜16日（右上に「前半」表示）
- 後半カード: 17日〜31日（左上に「後半」表示）
- 打刻: 出勤時間（左列）、退勤時間（右列）
- 手書き赤ペン: 有休、欠勤などの備考

### 変換ルール

#### 出勤時間の補正

- 実際の出勤時間が定時より早い → 出勤時間欄には定時を入力
- 実際の出勤時間が定時より遅い（遅刻） → 実際の出勤時間をそのまま入力
- 例: 定時8:30、実出勤7:48 → 出勤時間は8:30
- 例: 定時8:30、実出勤9:55 → 出勤時間は9:55

#### 退勤時間

- タイムカードの退勤打刻をそのまま入力

#### 有給

- タイムカードに赤ペンで「有休」「有給」と記載がある日
- CSV: 有給列に1
- 時給者: 有給時間列にマスターの有給時間（例: 8:00, 4:00, 3:30）を入力
- 月給者: 有給時間列は空

#### 欠勤（月給社員のみ）

- 水曜・日曜以外で出勤も有給もない日 → 欠勤扱い
- CSV: 欠勤列に1

#### 時給者の未出勤日

- 出勤がない日はCSVに行を作らない（欠勤フラグも不要）

#### 特殊ケース

- 松井嘉子: 定時13:00、カードに退勤のみ記録（出勤打刻なし）→ 出勤は全日13:00固定
- 松井慶信: 日によって13時台出勤と8時台出勤が混在 → 実際の出勤時間をそのまま入力（有給なし）
- 羽山寿男: 業務委託、一部手書き補正あり
