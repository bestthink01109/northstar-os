## 2. NorthStarの強み（競合が持たない／弱い領域）

### 2-1. 全カテゴリ共通の強み
- **LINE単体で業務完結**：競合は専用アプリ強要だが、当社は従業員の既存LINEで完結（導入摩擦ゼロ）
- **AI標準搭載（Claude）**：競合の多くはAIオプション扱い、または独自AI（精度低）
- **業務横断統合**：勤怠×LINE×経営レポートを1つのプラットフォームで提供
- **法改正の即時反映**：Claudeの知識ベース更新により、競合の「数週間後対応」を凌駕

### 2-2. KENZAI領域（建設）
- **現場員のITリテラシー不問**：LINE打刻なら高齢職人でも即日導入可能
- **元請・協力会社の壁を超える**：LINEは全員が持っているため、グリーンサイトのような業界専用網不要

### 2-3. LINE自動化領域
- **「マーケ用LINE」ではなく「業務用LINE」**：Lステップ等はBtoC集客特化、当社はBtoB業務効率化に振り切り
- **n8n統合による柔軟性**：他社は閉じたSaaS、当社は外部API/DBと自由に連携

### 2-4. 介護領域
- **社労士代行＋システム自動化のハイブリッド**：競合は「人 OR システム」二択、当社は「人＋AI」
- **多事業所経営者向け統合レポート**：単一事業所SaaSにない経営視点

## 5. 差別化メッセージ案（Jay Abraham視点）

### 5-1. 競合が見落としている「隠れた機会」
- **「現場の人」のITリテラシー断絶**：競合は全員「専用アプリを使える前提」で設計されているが、建設業の50代以上・介護業のパート層は事実上アプリ導入できない。LINEなら導入率100%。
- **「経営者の孤独」**：単一機能SaaSは現場担当者向け。多店舗・多事業所オーナーは「全部見える1枚」を渇望しているが、誰も提供していない。
- **「法改正の不安」**：社労士代行は数週間遅れ、SaaSは年1更新。AI即時反映は誰もやっていない。

### 5-2. NorthStarならではの価値（Jay Abraham「予防接種」型コピー）
> **「あなたの会社の勤怠管理、職人さんがアプリを覚えてくれないせいで、結局Excelに戻っていませんか？」**
>
> 多くの建設・介護経営者が、勤怠SaaSに月数万円払いながら、結局現場では紙とExcelに戻っています。理由はただひとつ──**現場の人がアプリを開かない**から。
>
> NorthStarは「LINEで打刻」する唯一のサービスです。新しいアプリは不要。ID・パスワードも不要。職人さん・介護スタッフが**今日から、何も覚えずに**使えます。
>
> しかも、給与計算・処遇改善加算・経営レポートまでAIが自動化。社労士費用と勤怠SaaS費用、両方を1本にまとめて、しかも**安く**。

### 5-3. 信頼できるアドバイザーになる根拠（Jay Abraham「Trusted Advisor」）
- **AI×人のハイブリッド**：「AIだけ」「人だけ」の競合に対し、当社は判断ミスをAIが下書き・人が監修する二重チェック
- **業界横断ノウハウ**：建設・介護・LINE自動化の3領域を運営する経営者視点
- **法改正即時反映の透明性**：「いつ・どの法改正に・どう対応したか」を顧客ダッシュボードに常時公開

# n8n Webhook経由でアップロード（環境変数 N8N_DRIVE_WEBHOOK_URL 必須）
if [ -z "${N8N_DRIVE_WEBHOOK_URL:-}" ]; then
  echo "[WARN] N8N_DRIVE_WEBHOOK_URL unset — falling back to manual workflow trigger" | tee -a "$LOG"
  bash /root/n8n-api.sh list | grep -i benchmark || true
  exit 0
fi

RESP=$(curl -sS -X POST "$N8N_DRIVE_WEBHOOK_URL" \
  -H "Content-Type: application/json" \
  -d "$(jq -nc \
        --arg path "$REPORT" \
        --arg folder "$FOLDER_ID" \
        '{path:$path, folderId:$folder}')")

echo "[$(date -Iseconds)] Drive upload response: $RESP" >> "$LOG"

# チケットを waiting に移動
mv "/root/northstar/tickets/doing/$TICKET" "/root/northstar/tickets/waiting/$TICKET" 2>/dev/null \
  || echo "[INFO] Ticket already moved or not in doing/" | tee -a "$LOG"

echo "[DONE] Benchmark uploaded and ticket handed off to Codex." | tee -a "$LOG"
