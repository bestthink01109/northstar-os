# Claude Code スキル候補一覧

patterns/ に登録されたパターンのうち、3回以上再利用されたものを

## スキル化フロー
```
パターン発生 → Type Cレポートに記録 → patterns/に登録 
→ 3回再利用 → ここに候補追加 → Claude Codeスキルとして定義
```

## スキル定義テンプレート

スキル化の際は以下のフォーマットで `~/.claude/skills/` に定義する:

```yaml
name: skill-name
description: |
  このスキルをいつ使うか（トリガー条件）。
  例: n8nワークフローのエラーをデバッグするとき。
steps:
  1. 確認事項1
  2. 確認事項2
  3. 解決手順
patterns:
  - patterns/n8n/README.md#パターン1
```
