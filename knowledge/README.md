# NorthStar OS ナレッジベース

日々の作業から得られた再利用可能な知識を蓄積するディレクトリ。

## 構成
- patterns/ - 解決済み問題のパターン集
- decisions/ - 重要決定事項の記録
- skills/ - Claude Codeスキル候補

## 蓄積ルール
1. レポートの「💡スキル化候補」セクションから抽出
2. COOが各セッション終了時にpatterns/またはdecisions/に登録
3. 同種の問題が3回発生したらskills/に昇格

## スキル化フロー
問題発生 → デバッグレポート(Type C)に記録 → patterns/に登録 → 3回再利用 → Claude Codeスキルとして定義
