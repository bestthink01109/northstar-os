## セールスレター22フォーミュラ

### Formula 1: こんな間違い（Common Mistake）
**最適用途：** 認知度がある業界向け・教育型アプローチ
**フロー：** 間違いリスト → 最大の間違い → 問題描写 → 商品 → ブレット7つ → オファー
```
{{アバター}}の方へ
あなたは{{テーマ}}でこんな間違いしていませんか？
・なぜ、{{間違い1}}してはいけないか知っていますか？
・なぜ、{{間違い2}}してはいけないか知っていますか？
・なぜ、{{間違い3}}してはいけないか知っていますか？
これらは、よくある間違いの一部です。でも、最大の間違いは{{最大の間違い}}です。
{{最大の間違いについての具体的な問題や悪夢のシーン}}。
それを解決するのが{{商品}}です。
{{ブレット7つ}}
{{行動喚起}}
```

### KENZAI（建設業勤怠→CSV）向けPAS
```
現場が増えるたびに、勤怠集計の手作業時間も倍になっていませんか？
多くの建設会社では、月20〜50時間が残業計算・出勤簿作成の手作業に消えています。
しかも法定内・法定外残業の振り分けミスは、給与計算全体に影響します。
「毎月この作業が来るたびに憂鬱になる」
「何度確認しても不安が拭えない」
こんな気持ちになったことはないですか？
建設業の複雑な勤怠計算は、避けて通れない業務です。
でも、自動化できます。
KENZAIは、出勤簿のExcel/PDFを投げるだけで残業計算・出勤簿作成・給与ソフト用CSVを自動生成します。

まず1ヶ月、無料でお試しいただけます。
```
